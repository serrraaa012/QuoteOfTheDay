## Quote of the Day (Android App)

A beautiful Android application that displays inspiring quotes daily with the ability to save favorites and share quotes.

## Features

- **Daily Quotes**: Displays a new inspiring quote each time you open the app
- **Quote Refresh**: Get a new random quote with the tap of a button
- **Favorite Quotes**: Save your favorite quotes to view later
- **Share Quotes**: Share inspiring quotes via messaging or social media
- **Favorites Management**: View and remove saved favorite quotes
- **Modern UI**: Beautiful gradient design with Material Design components

## Tech Stack

- **Language**: Java
- **Minimum SDK**: API 24 (Android 7.0 Nougat)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Database**: Room Database for local storage
- **UI Components**: Material Design, CardView, RecyclerView

## Libraries Used

- AndroidX AppCompat
- Material Design Components
- Room Database
- CardView
